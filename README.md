# Manual-Testing-Projects
Collection of Manual Test Cases, Bug Reports, and Test Plans for E-commerce and Web Applications.

## 🛠️ Tools Used
* **Jira:** Test management and bug tracking.
* **Chrome DevTools:** Inspecting elements and identifying UI issues.
* **GitHub:** Documentation and version control.

## 📋 Key Deliverables
* **Test Cases:** Covered 15+ scenarios (Positive/Negative/Edge Cases).
* **Bug Reports:** Documented critical UI layout issues and functional errors.
* **User Stories:** Created BDD-style stories with Acceptance Criteria.
