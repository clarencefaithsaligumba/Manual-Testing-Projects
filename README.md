# 🚀 Manual Testing Projects Portfolio

A comprehensive collection of manual testing documentation, bug reports, and test plans for e-commerce and web applications. This portfolio demonstrates my proficiency in ensuring software quality through meticulous testing methodologies.

---

## 📌 Projects Covered

### 1. Agile Travel Booking System
* **Description:** End-to-end functional testing of a flight booking application.
* **Focus:** Core transaction flow (Search -> Passenger Info -> Payment) and data consistency across pages.
* **Test Suite:** 23+ Test Cases covering Positive, Negative, and Edge Case scenarios.

### 2. Automation Exercise (E-commerce)
* **Description:** UI/UX and Functional testing of an e-commerce platform.
* **Focus:** Product search accuracy, cart management, and user registration workflow.

---

## 🛠️ Tools Used

* **Jira (Zephyr Scale):** Test case management, bug tracking, and execution monitoring.
* **Chrome DevTools:** Inspecting elements, identifying UI/UX issues, and monitoring console errors.
* **GitHub:** Version control and hosting of documentation.
* **MS Excel:** Structured test case reporting, prioritization, and execution tracking.

---

## 📋 Key Deliverables

* **Test Cases:** Detailed documentation of 40+ scenarios with clear Pre-conditions and Expected Results.
* **Bug Reports:** Professional bug logging including "Steps to Reproduce," "Expected vs. Actual Results," and "Severity/Priority" levels.
* **User Stories:** Agile-style stories created with clear **Acceptance Criteria (AC)** using BDD-style logic.
